🎯 핵심 기능
────────────────────────────────────────────────────────────────

8가지 업스케일 알고리즘:
  ✓ Nearest Neighbor (극도로 빠름)
  ✓ Bilinear (실시간)
  ✓ Bicubic (균형)
  ✓ Lanczos 4 (최고 품질)
  ✓ FSR - AMD (게임)
  ✓ NIS - NVIDIA (사진)
  ✓ xBR (픽셀아트)
  ✓ Real-ESRGAN (AI)

3가지 하이브리드 알고리즘 ⭐:
  ✓ IntelligentHybrid - 자동 분석 후 최적 선택
  ✓ MultiScaleUpscaler - 2×2×2 단계적 처리
  ✓ AdaptiveResourceUpscaler - 메모리 제약 자동 조정

고급 기능:
  ✓ 5가지 리소스 프로필 (극저~극고사양)
  ✓ 자동 시스템 감지
  ✓ 실시간 성능 모니터링
  ✓ 고급 GUI (tkinter)
  ✓ CLI 모드 (배치 처리)
  ✓ Python API

════════════════════════════════════════════════════════════════

🚀 3가지 실행 방식
────────────────────────────────────────────────────────────────

1️⃣ GUI (가장 쉬움)
   $ python pyscale.py
   → tkinter 기반 사용자 인터페이스 실행

2️⃣ CLI (빠름)
   $ python pyscale.py -i input.jpg -o output.png --scale 4
   → 커맨드라인 한 줄로 업스케일

3️⃣ Python API (프로그래밍)
   from src import IntelligentHybrid
   upscaler = IntelligentHybrid()
   result, metrics = upscaler.upscale_safe(image, 2.0)
