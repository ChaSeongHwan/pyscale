🚀 3가지 실행 방식
───────────────────────────────────────────────────────────────────

1️⃣ GUI (가장 쉬움)
   python pyscale_realtime.py
   
   기능:
   • 리소스 프로필 선택
   • 배율 조절
   • 알고리즘 선택
   • FPS 제한
   • 실시간 미리보기

2️⃣ CLI (고급)
   python pyscale_realtime.py --cli \
     --profile balanced \
     --scale 2.0 \
     --algorithm smart \
     --fps 60
   
   옵션:
   • --profile: 극저 ~ 극고사양
   • --scale: 배율 (1.0 ~ 4.0)
   • --algorithm: smart, bilinear, bicubic, lanczos, fsr, nis
   • --fps: 30 ~ 144
   • --region: x,y,width,height

3️⃣ Python API (라이브러리)
   from src.capture.realtime import RealtimeCapture
   from src.algorithms.hybrid import SmartHybrid
   
   capture = RealtimeCapture()
   upscaler = SmartHybrid(scale=2.0)
   # ... 커스텀 처리 ...
