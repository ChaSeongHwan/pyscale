🚀 시작하기
───────────────────────────────────────────────────────────────────

1. 의존성 설치
   $ pip install -r requirements.txt

2. GUI 실행
   $ python pyscale_realtime.py

3. CLI 실행
   $ python pyscale_realtime.py --cli --profile balanced

4. 예제 실행
   $ python examples/game_upscale.py
   $ python examples/stream_upscale.py
   $ python examples/api_example.py
